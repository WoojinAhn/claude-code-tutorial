# 슬래시 커맨드 치트시트

## 빠른 참조

| 커맨드 | 설명 | 사용 시점 |
|--------|------|----------|
| `/help` | 사용 가능한 모든 슬래시 커맨드 목록 표시 | 커맨드가 기억나지 않을 때 |
| `/status` | 현재 세션 정보 확인 (모델, 계정, 프로젝트) | 모델이나 세션 상태를 확인하고 싶을 때 |
| `/clear` | 대화 기록 완전 삭제 | 새 작업을 백지에서 시작하고 싶을 때 |
| `/cost` | 토큰 사용량과 예상 비용 확인 | 비용을 추적하고 싶을 때 |
| `/init` | 프로젝트용 CLAUDE.md 파일 생성 | 새 프로젝트에서 Claude Code를 처음 설정할 때 |
| `/memory` | CLAUDE.md 편집 | 프로젝트 컨텍스트나 컨벤션을 추가/수정할 때 |
| `/model` | AI 모델 전환 (opus, sonnet, haiku) | 작업 난이도에 맞는 모델로 바꾸고 싶을 때 |
| `/config` | Claude Code 설정 확인 및 변경 | 테마, 권한, 기본값 등을 변경할 때 |
| `/compact` | 대화를 요약해서 컨텍스트 윈도우 절약 | 대화가 길어져서 Claude가 앞부분을 잊기 시작할 때 |
| `/permissions` | 도구 권한 관리 (허용/거부/질문) | 매번 허용 클릭하는 게 귀찮거나, 특정 작업을 제한하고 싶을 때 |
| `/doctor` | Claude Code 환경 건강 체크 | 뭔가 안 될 때 원인을 진단하고 싶을 때 |
| `/vim` | vim 키바인딩 토글 | vim 스타일 편집을 사용하거나 해제할 때 |
| `/terminal-setup` | 터미널 통합 설정 안내 | 렌더링이 깨지거나 터미널을 최적화하고 싶을 때 |
| `/review` | 코드 리뷰 시작 | PR 전에 변경사항을 검토하고 싶을 때 |
| `/hooks` | 이벤트 훅 관리 | 특정 이벤트에 자동 동작을 연결하고 싶을 때 |
| `/mcp` | MCP 서버 관리 | 외부 도구나 데이터 소스를 연결하고 싶을 때 |
| `/login` | 인증 (로그인) | 인증이 필요하거나 계정을 전환할 때 |
| `/logout` | 인증 해제 (로그아웃) | 공유 환경에서 계정을 분리하거나 재인증이 필요할 때 |
| `/bug` | 버그 리포트 전송 | Claude Code에서 이상한 동작을 발견했을 때 |

## 카테고리별 정리

### 기본

| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/help` | 모든 커맨드 목록 보기 |
| `/status` | 세션 대시보드 |
| `/clear` | 대화 전체 삭제 (주의: 복구 불가) |
| `/cost` | 토큰 사용량 및 비용 확인 |

### 프로젝트 설정

| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/init` | CLAUDE.md 생성 (프로젝트 설정 문서) |
| `/memory` | CLAUDE.md 편집 |
| `/model` | 모델 전환 (opus/sonnet/haiku) |
| `/config` | 전체 설정 관리 |

### 컨텍스트 관리

| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/compact` | 대화 요약으로 컨텍스트 절약 |
| `/permissions` | 도구 권한 설정 |

### 개발자 도구

| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/doctor` | 환경 진단 |
| `/vim` | vim 모드 켜기/끄기 |
| `/terminal-setup` | 터미널 최적화 설정 |
| `/review` | 코드 리뷰 |

### 통합 기능

| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/hooks` | 이벤트 훅 관리 |
| `/mcp` | MCP 서버 관리 |
| `/login` | 로그인 |
| `/logout` | 로그아웃 |
| `/bug` | 버그 리포트 |

## 자주 쓰는 조합

| 상황 | 추천 커맨드 |
|------|-----------|
| 처음 시작 | `/help` → `/status` |
| 새 프로젝트 세팅 | `/init` → `/memory` → `/permissions` |
| 긴 코딩 세션 중 | `/compact` → `/cost` |
| 뭔가 안 될 때 | `/doctor` → `/status` |
| PR 전 점검 | `/review` |
| 모델 변경 | `/model` → `/status`로 확인 |
| 완전히 새로 시작 | `/clear` |
