# 슬래시 커맨드 치트시트

## 빠른 참조

| 커맨드 | 설명 | 사용 시점 |
|--------|------|----------|
| `/help` | 사용 가능한 모든 슬래시 커맨드 목록 표시 | 커맨드가 기억나지 않을 때 |
| `/status` | 현재 세션 정보 확인 (모델, 계정, 프로젝트) | 모델이나 세션 상태를 확인하고 싶을 때 |
| `/clear` | 대화 기록 완전 삭제 | 새 작업을 백지에서 시작하고 싶을 때 |
| `/init` | 프로젝트용 CLAUDE.md 파일 생성 | 새 프로젝트에서 Claude Code를 처음 설정할 때 |
| `/memory` | CLAUDE.md 편집 | 프로젝트 컨텍스트나 컨벤션을 추가/수정할 때 |
| `/config` | Claude Code 설정 확인 및 변경 | 테마, 권한, 기본값 등을 변경할 때 |
| `/permissions` | 도구 권한 관리 (허용/거부/질문) | 매번 허용 클릭하는 게 귀찮거나, 특정 작업을 제한하고 싶을 때 |
| `/model` | AI 모델 전환 (opus, sonnet, haiku) | 작업 난이도에 맞는 모델로 바꾸고 싶을 때 |
| `/plan` | 플랜 모드 진입 | Claude가 먼저 분석하고 계획을 세우길 원할 때 |
| `/vim` | vim 키바인딩 토글 | vim 스타일 편집을 사용하거나 해제할 때 |
| `/compact` | 대화를 요약해서 컨텍스트 윈도우 절약 | 대화가 길어져서 Claude가 앞부분을 잊기 시작할 때 |
| `/context` | 컨텍스트 윈도우 사용량 시각화 | 컨텍스트가 얼마나 남았는지 확인하고 싶을 때 |
| `/cost` | API 토큰 사용량과 비용 확인 (API 사용자) | API 과금 기반 사용 시 비용을 추적하고 싶을 때 |
| `/stats` | 사용 패턴, 스트릭, 모델 선호도 시각화 | 사용 습관을 파악하고 싶을 때 (구독자 권장) |
| `/resume` | 이전 세션 재개 | 어제 하던 작업을 이어가고 싶을 때 |
| `/rewind` | 대화와/또는 코드 되돌리기 | Claude가 잘못된 방향으로 갔을 때 |
| `/rename` | 현재 세션 이름 변경 | 나중에 이 세션을 쉽게 찾고 싶을 때 |
| `/copy` | 마지막 응답을 클립보드에 복사 | Claude의 응답을 다른 곳에 붙여넣고 싶을 때 |
| `/doctor` | Claude Code 환경 건강 체크 | 뭔가 안 될 때 원인을 진단하고 싶을 때 |
| `/terminal-setup` | 터미널 통합 설정 안내 | 멀티라인 입력 설정이나 터미널 최적화할 때 |
| `/theme` | 색상 테마 변경 | 다른 색상 테마를 사용하고 싶을 때 |
| `/review` | 코드 리뷰 시작 | PR 전에 변경사항을 검토하고 싶을 때 |
| `/mcp` | MCP 서버 관리 | 외부 도구나 데이터 소스를 연결하고 싶을 때 |
| `/hooks` | 이벤트 훅 관리 | 특정 이벤트에 자동 동작을 연결하고 싶을 때 |
| `/login` | 인증 (로그인) | 인증이 필요하거나 계정을 전환할 때 |
| `/logout` | 인증 해제 (로그아웃) | 공유 환경에서 계정을 분리하거나 재인증이 필요할 때 |
| `/bug` | 버그 리포트 전송 | Claude Code에서 이상한 동작을 발견했을 때 |

## 카테고리별 정리

### 기본
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/help` | 모든 커맨드 목록 보기 |
| `/status` | 세션 대시보드 |
| `/clear` | 대화 전체 삭제 (주의: 복구 불가) |

### 프로젝트 설정
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/init` | CLAUDE.md 생성 (프로젝트 설정 문서) |
| `/memory` | CLAUDE.md 편집 |
| `/config` | 전체 설정 관리 |
| `/permissions` | 도구 권한 설정 |

### 모델과 모드
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/model` | 모델 전환 (opus/sonnet/haiku) |
| `/plan` | 플랜 모드 — 분석 먼저, 실행 나중에 |
| `/vim` | vim 모드 켜기/끄기 |

### 컨텍스트와 사용량
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/compact` | 대화 요약으로 컨텍스트 절약 |
| `/context` | 컨텍스트 사용량 시각화 |
| `/cost` | API 토큰 비용 추적 (API 사용자) |
| `/stats` | 사용 패턴과 기록 확인 |

### 세션 관리
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/resume` | 이전 세션 이어가기 |
| `/rewind` | 이전 시점으로 되돌리기 |
| `/rename` | 세션 이름 붙이기 |
| `/copy` | 마지막 응답 클립보드 복사 |

### 개발자 도구
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/doctor` | 환경 진단 |
| `/terminal-setup` | 터미널 최적화 설정 |
| `/theme` | 색상 테마 변경 |
| `/review` | 코드 리뷰 |

### 통합 기능
| 커맨드 | 한 줄 설명 |
|--------|-----------|
| `/mcp` | MCP 서버 관리 |
| `/hooks` | 이벤트 훅 관리 |
| `/login` | 로그인 |
| `/logout` | 로그아웃 |
| `/bug` | 버그 리포트 |

## 자주 쓰는 조합

| 상황 | 추천 커맨드 |
|------|-----------|
| 처음 시작 | `/help` → `/status` |
| 새 프로젝트 세팅 | `/init` → `/memory` → `/permissions` |
| 긴 코딩 세션 중 | `/context` → `/compact` |
| 복잡한 작업 | `/plan` → 계획 검토 → `/model opus` |
| 잘못된 방향 | `/rewind` → 다른 접근 시도 |
| 뭔가 안 될 때 | `/doctor` → `/status` |
| PR 전 점검 | `/review` → `/copy` |
| 내일 이어서 하기 | `/rename` → 내일 `/resume` |
| 모델 변경 | `/model` → `/status`로 확인 |
| 완전히 새로 시작 | `/clear` |

## 주요 차이점

| | /clear | /compact |
|---|--------|---------|
| 대화 기록 | 삭제 | 요약 |
| 맥락 | 사라짐 | 보존 |
| 사용 시점 | 새 작업 | 같은 작업, 긴 세션 |

| | /init | /memory |
|---|-------|---------|
| 목적 | CLAUDE.md 생성 | CLAUDE.md 편집 |
| 사용 시점 | 처음 설정 | 지속적 보완 |

| | /cost | /stats |
|---|-------|--------|
| 대상 | API 사용자 | 모든 사용자 (구독자 권장) |
| 내용 | 토큰 비용, 과금 | 사용 패턴, 스트릭, 기록 |

| | 플랜 모드 | 일반 모드 |
|---|----------|----------|
| Claude 가능 작업 | 읽기, 분석, 계획 | 읽기, 쓰기, 실행 |
| 사용 시점 | 복잡한 작업, 접근 방식 검토 | 바로 변경 적용 |

## 키보드 단축키 (보너스)

| 단축키 | 기능 |
|--------|------|
| `Shift+Tab` / `Alt+M` | 권한 모드 순환 (일반 → 플랜 → 자동 수락) |
| `Esc Esc` | 되돌리기 (`/rewind`와 동일) |
| `Ctrl+C` | 현재 생성 취소 |
| `Ctrl+L` | 터미널 화면 정리 (대화는 유지) |
| `Ctrl+T` | 작업 목록 토글 |
| `!명령어` | bash 명령어 직접 실행 |
