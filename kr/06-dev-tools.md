# 모듈 6: 개발자 도구

이번 모듈에서는 개발 과정에서 유용한 유틸리티성 커맨드들을 다룹니다.

환경 문제 진단, 테마 변경, 코드 리뷰까지 — 일상적인 개발 워크플로우를 더 편하게 만들어주는 도구들입니다.

---

## 6.1 /doctor

**설명:** `/doctor`는 Claude Code의 건강 상태를 체크합니다. 설정 오류, 인증 문제, 환경 설정 이슈 등 일반적인 문제들을 자동으로 진단해줍니다. 뭔가 안 될 때 가장 먼저 실행해볼 커맨드입니다.

**문법:**
```
/doctor
```

**예시:** `/doctor`를 실행하면 여러 항목을 자동으로 검사합니다:
- 인증 상태 (API 키, 로그인 상태)
- 환경 설정 (Node.js 버전, 필수 의존성)
- 설정 파일 유효성
- 네트워크 연결 상태

문제가 발견되면 해결 방법도 함께 안내해줍니다.

**실습:** `/doctor`를 실행하고, 진단 결과에 **WARNING이나 ERROR가 있는지** 알려주세요. 전부 정상이면 "전부 OK"라고 답하면 됩니다.

**확인:** 결과를 확인합니다. 전부 OK면 건강한 환경입니다. 경고가 있다면 `Bash`로 관련 항목(git 버전, node 버전 등)을 직접 확인해서 추가 안내를 제공합니다.

**팁:** Claude Code가 갑자기 느려지거나, 이상한 에러가 나거나, 도구가 작동하지 않을 때 `/doctor`부터 돌려보세요. 대부분의 환경 문제는 여기서 원인을 찾을 수 있습니다.

---

## 6.2 /theme

**설명:** `/theme`은 Claude Code의 색상 테마를 변경합니다. 기본 테마가 마음에 안 들거나 터미널 환경에 맞지 않으면, 원하는 테마로 바꿀 수 있습니다.

**문법:**
```
/theme
```

**예시:** `/theme`을 실행하면 테마 선택기가 열립니다. 사용 가능한 테마를 탐색하고 선택할 수 있습니다. 테마 선택기 안에서 `Ctrl+T`를 누르면 코드 블록의 구문 강조 표시를 토글할 수 있습니다.

**실습:** `/theme`을 실행하고, 마음에 드는 테마를 골라서 **적용한 테마 이름**을 알려주세요. 언제든 다시 바꿀 수 있습니다.

**확인:** 테마 이름을 확인합니다. 변경은 즉시 적용됩니다.

**팁:** 작업 환경의 조명이 바뀌면 (밝은 사무실 vs 어두운 방) 테마도 맞춰 바꾸면 눈이 편합니다. 밝은 터미널에는 라이트 테마, 어두운 터미널에는 다크 테마가 잘 맞습니다.

---

## 6.3 /review

**설명:** `/review`는 코드 리뷰를 시작합니다. Claude가 현재 작업 디렉토리의 Git 변경사항을 분석해서 잠재적 문제, 코드 스타일, 개선점을 짚어줍니다. PR을 올리기 전에 셀프 체크로 활용하면 효과적입니다.

**문법:**
```
/review
```

**예시:** `/review`를 실행하면 Claude가 현재 변경사항을 분석합니다:
- 잠재적 버그나 에지 케이스 발견
- 코드 품질 및 가독성 피드백
- 보안 관련 주의사항
- 개선 제안

"보안 위주로 리뷰해줘"나 "에러 핸들링 위주로 봐줘"처럼 초점을 지정할 수도 있습니다.

**실습:** 이 실습은 3단계로 진행합니다. (Git 리포지토리가 아니라면 이 실습은 건너뜁니다.)

> **Step 1:** 튜터가 `Edit` 도구를 사용해서 프로젝트 내 파일에 리뷰 거리가 될 작은 변경(미사용 변수 추가, 에러 핸들링 누락 등)을 만듭니다. 유저에게 어떤 파일에 무엇을 수정했는지 알려줍니다.
>
> **Step 2:** 유저에게 `/review`를 실행하라고 안내합니다.
>
> **Step 3:** 리뷰 결과에서 Claude가 지적한 항목이 있는지 하나를 골라서 알려달라고 합니다.

**확인:** Claude의 리뷰는 잠재적 버그, 스타일, 보안 등을 카테고리별로 짚어줍니다. 실제 PR 전에 돌려보면 사람 리뷰어가 보기 전에 명백한 실수를 잡을 수 있습니다.

**팁:** `/review`는 사람 리뷰를 대체하는 게 아니라 보완하는 도구입니다. 기계적인 체크(누락된 에러 핸들링, 일관성 등)는 Claude에게, 설계와 로직 판단은 팀원에게 맡기세요.
